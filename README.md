# 📰 데이터로 파헤친 ‘가짜 후기’의 민낯

온라인 블로그·카페 게시글과 댓글 데이터를 수집해 후기형 광고, 스텔스 마케팅 의심 신호, 댓글 기반 구매 유도 구조를 분석한 데이터저널리즘 프로젝트입니다. 최종 산출물은 정적 웹페이지 형태의 탐사보도 기사입니다.

## 📌 프로젝트 개요

이 프로젝트는 의약품·건강기능식품 관련 온라인 후기에서 광고와 실제 경험담의 경계가 어떻게 흐려지는지 확인하기 위해 진행했습니다.

분석 대상은 다이어트, 비염, 키성장, 탈모, 피부미용 5개 주제입니다. 네이버 검색 API를 활용해 블로그·카페 게시글 후보를 수집하고, URL 기준 중복 제거 후 원문과 댓글을 보강했습니다. 이후 자체 설계한 `스텔스 마케팅 의심 신호 지표`와 LLM 분석을 결합해 게시글의 검토 필요도를 분류했습니다.

## ❓ 주요 질문

- 온라인 후기형 게시글에서 광고성 의심 신호는 얼마나 자주 나타나는가?
- 블로그와 카페는 검토 필요도 분포에서 차이를 보이는가?
- 댓글은 실제 소비자 반응에 가까운가, 아니면 구매 유도 흐름을 강화하는가?
- 특정 분야, 특히 탈모 관련 댓글에서 어떤 단어와 의미 연결 구조가 나타나는가?
- 소비자가 후기형 광고를 비판적으로 읽기 위해 확인해야 할 기준은 무엇인가?

## 🔍 분석 방법

### 1. 게시글 수집 및 정제

네이버 검색 API로 5개 주제별 키워드 조합을 검색했습니다. 검색 결과에서 제목과 요약문에 일정 개수 이상의 키워드가 포함된 게시글을 후보로 남겼습니다. 이후 같은 URL이 여러 키워드 조합에서 반복 수집되는 문제를 줄이기 위해 URL 기준으로 중복 제거를 수행했습니다.

### 2. 본문·댓글 보강

검색 API가 제공하는 데이터는 제목과 요약문 중심이기 때문에, 수집된 URL을 기준으로 접근 가능한 게시글의 본문을 추가 수집했습니다. 댓글은 별도 CSV로 저장하고, 게시글 데이터에는 댓글 수집 상태와 수집 댓글 수를 요약 컬럼으로 병합했습니다.

### 3. LLM 기반 게시글 분석

자체 설계한 `스텔스 마케팅 의심 신호 지표`를 시스템 프롬프트로 구성해 게시글 원문을 분석했습니다. 이 지표는 후기·체험담 형식, 소비자 추천 표현, 광고 표시 불분명성, 과장 표현, 조건부 대가성 표현 등 13개 항목으로 구성했습니다.

LLM 분석 결과는 다음과 같은 검토 분류로 정리했습니다.

| 분류 | 의미 |
| --- | --- |
| 검토 우선 | 의심 신호가 많이 감지되어 우선 확인이 필요한 게시글 |
| 검토 필요 | 일부 의심 신호가 감지되어 추가 확인이 필요한 게시글 |
| 검토 낮음 | 의심 신호가 적게 감지된 게시글 |
| 정상 광고 | 광고·협찬 표시가 명확한 게시글 |

### 4. 댓글 텍스트 분석

댓글 데이터는 한국어 불용어를 제거한 뒤 단어 빈도와 동시출현 관계를 분석했습니다.

- 주제별 댓글 워드클라우드
- 5개 주제 통합 워드클라우드
- 댓글 의미연결망
- 탈모·다이어트 주제의 검토분류별 댓글 비교
- 댓글 내 URL 언급 사례 탐색

## 📰 웹페이지 구성

최종 기사는 순수 HTML/CSS/JavaScript로 제작했습니다. React, Vue, Vite 등 프레임워크는 사용하지 않았습니다.

```text
├── index.html
├── index.css
├── index.js
└── assets/
    ├── photo_01.png
    ├── ...
    └── photo_10.png
```

시각화는 기존 HTML 그래프를 iframe으로 삽입하지 않고, 필요한 데이터와 렌더링 코드를 `index.js`에 통합했습니다. 차트 렌더링에는 D3, d3-cloud, vis-network를 사용했습니다.


## ⚠️ 한계

- LLM 분석 결과는 법적 판정이 아니라 텍스트 기반 검토 필요도 분류입니다.
- 실제 광고주 지시, 대가 지급 여부, 계약 관계는 별도 취재 없이는 확정할 수 없습니다.
- 접근 제한이 있는 카페 게시글은 본문과 댓글 수집이 제한될 수 있습니다.
- 댓글의 단어 빈도와 의미연결망은 광고 여부를 직접 증명하지 않으며, 반복 패턴을 해석하기 위한 보조 지표입니다.
